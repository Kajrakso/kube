# kube

An optimal solver for the 3x3x3 Rubik's cube.

*Work in progress.*

### Usage

Run

```sh
make
```

to build the project and 

```sh
make test
```
to run the tests (you need [Criterion](https://github.com/Snaipe/Criterion) for these).
